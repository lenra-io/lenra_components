LenraTable

LenraTable class Null safety

Lenra implementation of Flutter's Table widget.

The children of this widget should be of type LenraTableRow. The centerChildren property is used to vertically and horizontally center each child of the table. The border property is used to show a thin black border around each cell of the table.

In case it is needed to put an empty LenraTableCell in the Table, just set a null child in the LenraTableRow. Example :

LenraTable(
  children: [
    LenraTableRow(
      children: [
        Text("example"),
        null, // This will show an empty cell
      ],
    ),
  ],
),
